Belmont provides a structured workflow for planning, implementing, and verifying product work.

Skill index (files in this directory):
- `product-plan.md` Create the PRD and milestone plan.
- `tech-plan.md` Produce the technical implementation plan after the PRD exists.
- `implement.md` Implement the next milestone end to end.
- `next.md` Implement the next single task only.
- `verify.md` Run verification and code review.
- `status.md` Summarize current progress.
- `reset.md` Reset Belmont state files.

When a user requests a Belmont workflow, load the matching file above and follow its instructions as the source of truth for the task.
