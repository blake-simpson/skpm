Available skills:
- `belmont` Multi-step product planning and implementation workflow. Files live in `.codex/belmont/`.

Commands (use exactly these slash commands in chat):
- `/belmont:product-plan`
- `/belmont:tech-plan`
- `/belmont:implement`
- `/belmont:next`
- `/belmont:verify`
- `/belmont:status`

Notes:
- These are **skills**, not shell commands. Do **not** run `belmont:status` in the terminal.
- If you need the CLI, run `belmont status` (space), not `belmont:status`.
- Other helpers: `belmont tree`, `belmont find`, `belmont search`.
